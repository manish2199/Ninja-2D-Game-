# Ninja-2D-Game
  2D Platformer game
  
 
# Features
1.SinglePlayer

2.Multiplayer

3.Smooth Player Movement

4.Gliding and Sliding Ability.

5.Sharp Shooting Mechanism.

6.Different Traps.

7.Beautiful Level Design.

8.Unique Wall Climbing Technique.
